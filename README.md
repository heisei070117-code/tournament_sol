# Bracket Studio

シード抽選・手動配置・勝ち上がり管理に対応した、シングルエリミネーション専用のトーナメント表作成アプリです。すべてブラウザ内で動作し、入力内容は外部へ送信されません。

## 主な機能

- 2～256名、任意シード数、公平に分散したBYEの自動処理
- BYE試合を省略し、必要な予備戦だけを表示する自然なラウンド構成
- 標準シード配置を使ったランダム抽選
- シード番号ごとに現在のチームが分かる割り当て画面
- 全枠の手動配置と参加者の自動入れ替え
- 勝者クリック、次ラウンド反映、チャンピオン表示
- 対戦する2チームから個別に伸びて合流し、確定した勝者ルートだけを赤色で示す標準的なトーナメント線
- ブラウザ内への自動保存
- 編集可能なJSONの保存・読込
- SVG出力、印刷、PDF保存
- PC・スマートフォン対応

詳細な要件と受入条件は [SPECIFICATION.md](SPECIFICATION.md) を参照してください。

## 起動方法

依存パッケージのインストールは不要です。

```bash
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080` を開いてください。単純な確認であれば `index.html` を直接開くこともできます。

## テスト

Node.js 18以降で実行します。

```bash
npm test
```

## ファイル構成

```text
index.html                 画面構造
styles.css                 レスポンシブUI・印刷スタイル
src/app.js                 UI、保存、ファイル入出力
src/tournament.js          抽選・シード・勝ち上がりロジック
tests/tournament.test.js   ロジックの単体テスト
SPECIFICATION.md           実装仕様書
```

## データについて

大会データはブラウザの `localStorage` に保存されます。ブラウザのサイトデータを削除すると消えるため、長期保存や別端末への移行にはJSON保存を利用してください。
